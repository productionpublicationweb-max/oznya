🚀 Installation

Code d'intégration de base

Ajoutez ce code avant la balise </body> de votre page :

<script src="https://votre-domaine.com/widget/nyxia-widget.js" defer></script>

Avec des options personnalisées

<script src="https://votre-domaine.com/widget/nyxia-widget.js" data-title="Mon Assistante" data-position="bottom-left" data-primary-color="#9D4EDD" data-auto-open="false" defer ></script>

⚙️ Options de configuration

Attribut Description Défaut
data-title Titre affiché dans l'en-tête du widget "Nyxia"
data-position Position du bouton : "bottom-right" ou "bottom-left" "bottom-right"
data-theme Thème visuel : "dark" ou "light" "dark"
data-primary-color Couleur principale (hexadécimal) "#00D4FF"
data-secondary-color Couleur secondaire (hexadécimal) "#9D4EDD"
data-auto-open Ouvrir automatiquement le widget au chargement "false"
data-chat-url URL personnalisée du chat Domaine actuel

🎮 API JavaScript

Le widget expose une API JavaScript pour contrôler son comportement :

// Ouvrir le widget window.NyxiaWidget.open(); // Fermer le widget window.NyxiaWidget.close(); // Basculer l'état ouvert/fermé window.NyxiaWidget.toggle(); // Vérifier si le widget est ouvert if (window.NyxiaWidget.isOpen()) { console.log('Le widget est ouvert'); }

📦 Intégration dans les coffrets

Pour intégrer Nyxia dans vos coffrets numériques, vous pouvez :

Option 1 : Lien direct

Incluez le script du widget dans n'importe quelle page HTML de votre coffret numérique.

⚠️ Important : Assurez-vous que le domaine hébergeant le widget est accessible publiquement pour vos clients.

❓ FAQ

Le widget fonctionne-t-il sur tous les sites ?

Oui, le widget est compatible avec tous les sites web, qu'ils soient statiques, WordPress, Shopify, Wix, etc.

Puis-je personnaliser l'apparence ?

Absolument ! Utilisez les attributs data-primary-color et data-secondary-color pour adapter les couleurs à votre marque.

Comment puis-je suivre les conversions ?

Les liens de services incluent automatiquement des paramètres UTM pour le suivi dans Google Analytics ou votre outil d'analyse préféré.
